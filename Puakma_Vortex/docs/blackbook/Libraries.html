Libraries

Libraries may be either single java class files or multiple class files in a jar. Libraries provide a way to share reusable code amongst your actions and Business Widgets.

To create a library:

  1. Create and compile a new Java class
  2. Open the libraries section of the webdesign application and click the “New Library” button
  3. Give the library a name
  4. Attach the binary class file and optionally the source

All libraries are automatically loaded when an action is executed for the first time. ${TITLE}
